# System Health Monitor 🚀

A simple and efficient Bash script to monitor your system's vital signs.

## Features
- 💾 **Disk Usage:** Check available space on your partitions.
- 🧠 **RAM Monitor:** Real-time memory usage stats.
- ⚡ **CPU Load:** View current system load averages.
- 🔔 **Reboot Alerts:** Notifications if the system needs a restart.

## How to use
1. Give execution permissions:
   `chmod +x system_check.sh`
2. Run the script:
   `./system_check.sh`
